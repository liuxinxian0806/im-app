# 「我的」模块功能规格说明书 (原始需求)
我的列表是一系列的功能入口列表。在其右侧区域展示各个入口点击后的内容
## 0. 我的列表
*   **个人资料展示入口**：一级布局：头像（方型、四周稍有圆角）、昵称、BIBI号、右箭头图标。整体水平布局，头像在左侧。昵称、BIBI号在头像右边，最右侧右箭头图标。头像展开后的二级页面布局：头像、昵称、BIBI号、地区、二维码；支持跳转个人资料编辑页。整体水平布局，头像在左侧。昵称、BIBI号、地区在头像右边，二维码靠右上角。
*   **我的钱包入口**
*   **我的名片入口**：
*   **名片夹入口**
*   **收藏入口**
*   **我的货盘入口**
*   **我发布的招聘入口**
*   **我的订单入口**
*   **设置入口**
## 1. 个人资料
*   **个人资料展示**：一级布局：头像（方型、四周稍有圆角）、昵称、BIBI号、右箭头图标。整体水平布局，头像在左侧。昵称、BIBI号在头像右边，最右侧右箭头图标。
*   **点击箭头后展示内容**：
    * 最顶部：头像（方型、四周稍有圆角）、昵称、BIBI号、地区、二维码；支持跳转个人资料编辑页。整体水平布局，头像在左侧。昵称、BIBI号、地区在头像右边，二维码靠右上角。
    * 中间部分：两个元素，一左一右两端对齐，例如第一组左侧文案：头像，右侧头像展示，点击选图替换头像；第二组左侧文案：昵称，右侧昵称展示，点击编辑替换昵称
    * 最底部：按钮「编辑资料」

## 2. 我的钱包
*   **展示内容**：
    *   展示我的余额：xxx积分=$xxx，最终转换成人民币（提现金额为人民币）
    *   提示支持提现到微信钱包或支付宝
*   **交互逻辑**：点击【提现】跳转至余额提分页
*   **提现页面功能**：
    *   支持选择提现渠道（提现到微信钱包或支付宝）
    *   提现余额（展示当前可提现余额，支持一键全部提现）
    *   提现规则说明及常见问题（支持联系客服-拉起客服聊天IM）
*   **账单明细**：
    *   余额明细：按月展示余额明细 - 展示所有获取与支出明细；获取明细来源运价发现，需展示该订单号

## 3. 我的名片
*   **展示状态**：
    *   已完成制作的完整名片展示（展示内容待定）
    *   未开始进行制作编辑的名片展示（空白）
*   **功能模块**：
    *   名片编辑：支持选择不同的名片样式（待UI设计）
    *   分享文案：支持自定义
*   **字段要求**：
    *   名片图（必填）
    *   姓名（必填）：必须填写真实姓名
    *   手机号：按注册登录绑定的手机号展示，支持直接修改名片的手机号（修改的手机号不影响BIBITALK绑定的手机号）
    *   邮箱（非必填）：支持直接输入
    *   公司名称（必填）：需满足企查查认证的公司名称，否则不允许填入
    *   公司logo（非必传）
    *   公司地址（非必填）
    *   优势航线、合作船东：支持选择（最多各选择3项）

## 4. 名片夹
*   **名片收录**：收录所有好友的名片（添加好友时，自动收到好友的名变，将全部收纳此处）；其中收藏的名片带星号标识展示
*   **收藏名片**：收录所有收藏的名片；支持取消收藏
*   **交互逻辑**：点击名片 $\rightarrow$ 进入他的名片主页

## 5. 我的货盘
*   **展示内容**：展示我发布的货盘卡
*   **按钮功能**：【再次发布】
*   **状态标识**：包含询盘中 / 已成交 / 已完成（货盘已过期）
*   **交互逻辑**：
    *   发布者点击空白处 $\rightarrow$ 跳转至货盘详情页，支持确认货盘成交 / 关闭货盘 / 再次发布
    *   点击【再次发布】 $\rightarrow$ 跳转至货盘发布编辑页，带入原始数据，支持修改编辑，重新发布

## 6. 我发布的招聘
*   **展示内容**：展示我发布的招聘卡
*   **按钮功能**：【再次发布】
*   **状态标识**：包含进行中 / 已完成（招聘信息已到期）/ 已关闭
*   **交互逻辑**：
    *   发布者点击空白处 $\rightarrow$ 跳转至招聘详情页，支持关闭招聘 / 再次发布
    *   点击【再次发布】 $\rightarrow$ 跳转至招聘发布编辑页，带入原始数据，支持修改编辑，重新发布

## 7. 收藏
*   **分类展示**：全部收藏内容（包含图片与视频、聊天记录、文件）
*   **具体项功能**：
    *   图片与视频：支持左滑取消某条收藏；点击图片跳转至详情页，详情页点击图片直接放大
    *   聊天记录：包含与好友单条/多条聊天内容或群聊记录的收藏；点击进入对应聊天详情页
    *   文件：收录收藏的文件；支持预览（未下载提示下载）和文件预览页查看
*   **操作**：详情内支持取消该收藏

## 8. 我的订单
*   **展示内容**：展示我的运价订单卡，展示订单号，展示本单返佣金额
*   **按钮功能**：【再次购买】
*   **交互逻辑**：
    *   预定者点击卡片 $\rightarrow$ 进入订单详情页，支持联系客服 / 再次购买
    *   点击【再次购买】 $\rightarrow$ 进入该运价的详情页；若该订单已关闭/已完成，则“预定下单”按钮置灰不可用

## 9. 设置
*   **设置列表内容**：个人资料、账号与安全、接收新消息通知、朋友权限、帮助与反馈、关于BIBITALK、当前版本
*   **子模块功能**：
    *   **个人资料**：跳转至个人资料列表页，支持头像/昵称/BIBI号/性别/地区/个性签名修改；BIBI账号仅支持一年修改一次
    *   **账号与安全**：支持绑定手机号修改 / 支持BIBI账号修改 / 支持设置登录密码（若已有密码，支持修改）/ 支持注销BIBI账号
    *   **朋友权限**：
        *   好友申请权限：底部弹框拉起申请权限勾选（单选）- 同意任何用户加好友 / 需要验证 / 拒绝任何人加好友 / 取消
        *   仅聊天：跳转至仅聊天的用户列表
        *   朋友圈：设置可见范围（不让他看 / 不看他 / 允许朋友查看最近三天/一个月/半年/全部）
    *   **黑名单**：通讯录黑名单列表
